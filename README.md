# Shangban

一个 web 版本的哄自己上班软件。它用独立三花猫桌宠、翻页倒计时和可交互猫叫，帮你更温和地度过工作时间。

## 功能

- 独立透明三花猫桌宠形象
- 空闲呼吸、靠近好奇、点击伸爪、双击开心、长按呼噜
- 不同互动触发不同真实猫叫声
- 上班前、上班中、下班后的翻页倒计时
- 本地读取上下班时间设置
- Netlify 静态部署配置

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

Netlify 构建命令为 `npm run build`，发布目录为 `dist`。
